# 美味伴侣 - 每日菜单规划系统

为心爱的她精心规划每一餐的贴心应用。

## 功能特性

- 🍽️ **三餐管理**：分别管理早餐、午餐、晚餐
- 📅 **日期选择**：按日期规划菜单，支持前后翻页
- 🎯 **智能推荐**：丰富的菜品库，按类别筛选
- 🔍 **搜索功能**：快速查找喜爱的菜品
- 💾 **数据持久化**：本地存储，数据永不丢失
- 📱 **响应式设计**：适配各种屏幕尺寸
- ✨ **炫酷动效**：流畅的交互动效

## 使用方法

1. 选择日期规划菜单
2. 点击各餐食的"+"按钮添加菜品
3. 从弹出的菜品库中选择喜爱的菜品
4. 随时删除不需要的菜品
5. 所有数据自动保存到本地

## 技术栈

- HTML5
- CSS3 (包含Flexbox/Grid、渐变、动画)
- JavaScript (ES6+)
- localStorage (数据持久化)

## 系统要求

支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
