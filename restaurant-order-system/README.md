# 餐厅点餐系统

一个功能完善、界面友好的餐厅数字化点餐系统，为顾客提供便捷的自助点餐体验，同时为餐厅提供高效的订单管理功能。

## 功能特色

### 点餐功能
- **分类浏览** - 按类别浏览菜单（开胃菜、主菜、甜品、饮品）
- **菜品详情** - 显示菜品名称、价格、详细描述
- **便捷添加** - 点击菜品即可加入订单
- **数量调整** - 支持增加或减少菜品数量
- **实时总价** - 自动计算订单总金额

### 订单管理
- **购物车预览** - 实时查看当前订单
- **订单编辑** - 可修改菜品数量或删除菜品
- **订单清空** - 一键清空当前订单
- **桌号登记** - 输入桌号便于厨房配送
- **人数统计** - 记录用餐人数信息

### 用户体验
- **响应式设计** - 适配桌面端和移动设备
- **流畅交互** - 直观的操作界面和动画反馈
- **即时通知** - 操作成功提示反馈
- **视觉设计** - 现代化UI风格，美观易用

### 技术特点
- **本地存储** - 无需服务器即可运行
- **模块化架构** - 便于功能扩展和维护
- **数据模拟** - 内置示例菜单数据
- **表单验证** - 确保订单信息完整性

## 使用说明

1. 选择菜品分类浏览菜单
2. 点击菜品将其添加到订单
3. 在订单区域调整菜品数量
4. 输入桌号和用餐人数
5. 确认订单信息后点击"提交订单"
6. 厨房将收到订单并开始准备

## 业务价值

- **提高效率** - 减少服务员点餐时间
- **减少错误** - 顾客自主点餐避免沟通误解
- **降低成本** - 减少人力成本
- **改善体验** - 顾客可从容选择菜品
- **数据分析** - 便于收集销售数据
