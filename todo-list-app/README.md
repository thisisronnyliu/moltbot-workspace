# 简约待办事项清单 (Minimal Todo List)

一个美观、功能齐全且交互性强的纯前端待办事项应用。

## 特性

- ✨ 现代简约的设计风格
- 📱 响应式布局，支持移动设备
- 💾 本地存储，数据持久化
- 🏷️ 任务过滤（全部/未完成/已完成）
- 📊 实时统计数据
- ✏️ 任务编辑功能
- 🎨 流畅的动画和过渡效果
- 🔄 任务拖拽排序（可选功能）
- 🎯 键盘快捷键支持

## 技术栈

- HTML5
- CSS3 (包含渐变、阴影、动画等现代特性)
- JavaScript (ES6+)
- localStorage (数据持久化)

## 功能说明

1. **添加任务**: 在输入框输入任务后点击"添加"或按回车键
2. **标记完成**: 点击任务前的复选框
3. **编辑任务**: 点击"编辑"按钮修改任务内容
4. **删除任务**: 点击"删除"按钮
5. **过滤任务**: 使用顶部的过滤按钮切换视图
6. **统计数据**: 实时显示总任务数、已完成数和剩余数
7. **批量操作**: 清除已完成任务或清空全部任务

## 使用方法

1. 直接打开 `index.html` 文件即可使用
2. 所有数据都保存在浏览器的本地存储中
